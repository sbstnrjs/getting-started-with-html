The Web

The major pieces of the web are the computer and the browser, the Internet itself, and the servers that reply to the requests that the user (the computer and the browser) does.

The world wide web is built using servers that talk to each other. One computer places a request, another serves up the content. HTML is the language used to decode or read the request placed by the computer. On the receiving end, a browser can display the results.

HTML

HTML, (Hypertext Markup Language.) is made up of:
Text Content: what you see
Markup: what it looks like
References to other documents: like downloads, images or videos.
links to other pages: clickable links to images, files, or other locations on the web.

Tags and Elements

An HTML document is made up of elements, each of these elements or tags, dictate the way the browser displays the page. Some examples of tags are the b (bold) tag or the em (emphasis) tag.

Void Tags

A void tag is a tag that has no content and it needs no closing tag.
